# Dungeon Crawler

This project is a dungeon crawler with procedural generation written in Java.


## Game mechanics

* Rooms and corridors
* Monsters and bosses
* Items with enchantements
    - swords
    - bows
    - shields
    - hats
    - potions
* 3 difficulties
