# Dungeon Crawler

This project is a dungeon crawler with procedural generation written in Java.


## Game mechanics

* Rooms and corridors
* Monsters and bosses
* Items
    - swords
    - bows
    - shields
    - hats
    - potions
* Enchanted items
* 3 difficulties
